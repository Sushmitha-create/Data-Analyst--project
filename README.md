# 🚗 Traffic Accident Analysis & Severity Prediction

An interactive dashboard built with Streamlit to analyze 
traffic accident data and predict accident severity using 
Machine Learning.

## 📌 Project Overview
- Internship project at Skyllx Technologies Pvt Ltd
- Analyzed 589,340 accident records
- Achieved 86.76% accuracy with Random Forest Classifier

## 🛠️ Technologies Used
- Python, Pandas, NumPy
- Matplotlib, Seaborn
- Scikit-learn (Random Forest)
- Streamlit

## 📊 Features
- Interactive filters (weather, severity, lighting, time)
- KPI metrics dashboard
- Multiple visualizations (bar, pie, line charts)
- ML severity prediction with confusion matrix

## 🚀 How to Run
pip install -r requirements.txt
streamlit run app.py

## 📁 Dataset
Dataset: Chicago Traffic Crashes (CSV format)
[Download from Kaggle]
